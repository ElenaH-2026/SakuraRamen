<div align="center">
        
**Sakura Ramen** ist eine moderne, vollständig responsive Landingpage, die basierend auf präzisen Figma-Design-Richtlinien optimiert wurde.
<br>Die Webseite bietet eine interaktive Menüpräsentation sowie eine flüssige, benutzerfreundliche Navigation zu den wichtigsten Sektionen.
<br>Durch einen direkt nutzbaren Click-to-Call-Button im Header und eine integrierte Google-Maps-Karte zur schnellen Standortsuche ist eine gute Kundeninteraktion gewährleistet.

## 🚀 Quickstart

1. Repository klonen:

```bash
git clone https://github.com/ElenaH-2026/SakuraRamen.git
```

2. In das Verzeichnis wechseln:

```bash
cd SakuraRamen
```

3. Starte deinen lokalen Entwicklungsserver:
```text
z.B. Live Server in VS Code
```

4. Öffne danach die URL im Browser auf deinem Computer:
```text
http://localhost:<live-server-port>
```

5. Oder öffne die URL im Browser auf deiner virtuellen Maschine:
```text
http://<deine_ip>:<live-server-port>
```

### 🌟 Features

| Feature | Zusatzinformation |
| --- | --- |
| 📐 Figma-konformes Wireframe-Layout | *Das visuelle Design und die strukturelle Anordnung der App basieren exakt auf der vorgegebenen Design-Richtlinie für Tablet- und Mobil-Layouts.* |
| 📱 Vollständig Responsive | *Die Benutzeroberfläche ist mittels Media Queries von 320px über Tablet-Größen bis hin zu Desktop-Monitoren optimiert.* |
| ⚓ Flüssige Navigation (Smooth Scroll) | *Die Verlinkungen im Header erlauben ein nahtloses und geschmeidiges Scrollen zu den verschiedenen Sektionen wie dem Menü oder den Standorten.* |
| 🍜 Menü-Sektion | *Präsentation der Gerichte in einer Flexbox, die sich je nach Bildschirmgröße intelligent anordnet.* |
| 📞 Direktwahl-Button (Click-to-Call) | *Ein im Header integrierter, markanter Button ermöglicht mobilen Nutzern den direkten telefonischen Kontaktaufbau per Klick.* |
| 📍 Integrierte Google Maps Karte | *Einbettung einer interaktiven Google Maps Karte via iFrame zur genauen Lokalisierung des Restaurants.* |
| 🌐 Social-Media-Anbindung | *Direkte Verknüpfungen im Footer-Bereich zu relevanten Profilen auf TripAdvisor, Facebook und Instagram zur Steigerung der Online-Präsenz.* |
